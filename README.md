## Hi there 👋

ToolAuditDE ist ein Community-basiertes Prüf- und Zertifizierungssystem für Third-Party-Tools wie RTSS, Magpie, 
Lossless Scaling, Upscaler, Overlays und weitere Software, die in Spielen oder produktiven Umgebungen eingesetzt wird. Ziel ist es, Transparenz, 
Sicherheit und Vertrauen zwischen Nutzern, Entwicklern und Herstellern zu schaffen.

Das offizielle ToolAudit‑Siegel kennzeichnet Tools, die durch das ToolAudit‑Team geprüft und als sicher eingestuft wurden.
Es besitzt nur dann Gültigkeit, wenn der Entwickler im internen ToolAudit‑Discord als Aktiv geführt wird.
Das Siegel zeigt an, dass das Tool die technischen und organisatorischen Anforderungen des Projekts erfüllt.

Das Community‑Siegel ist ein rein visuelles Icon, das von der Community genutzt werden kann.
Es hat keine offizielle Zertifizierungsfunktion und dient ausschließlich als Symbol für Unterstützung oder positive Resonanz.
Ein Tool gilt nicht als offiziell zertifiziert, nur weil dieses Icon verwendet wird.


5. Zertifizierungsprozess
Entwickler reicht Tool ein

ToolAudit‑Team prüft das Tool

Prüfer erstellen einen technischen Bericht

Zertifizierer entscheiden über SAFE / PASSIVE / LIMITED / UNSAFE

Entwicklerstatus im Discord bestimmt die Gültigkeit

Zertifikat muss bei größeren Updates erneuert werden

6. Entwicklerstatus im Discord
Ein Tool ist nur dann offiziell gültig, wenn der Entwickler im internen ToolAudit‑Discord als:

Aktiv

Verifiziert

Zertifiziert

geführt wird.
Ohne aktiven Entwicklerstatus verliert das Siegel automatisch seine Gültigkeit.

7. Roadmap
Phase 1: Geschlossener Team‑Discord (jetzt)

Phase 2: Entwickler‑Einladungen

Phase 3: Erste Tools prüfen

Phase 4: Veröffentlichung erster Zertifikate

Phase 5: Community‑Öffnung in 2–3 Jahren

Phase 6: Ausbau der Prüfer‑ und Zertifizierer‑Teams

8. Mitmachen
ToolAudit ist ein kostenloses Projekt.
Aktuell ist der Discord nur für Team und Entwickler geöffnet.
Später können folgende Rollen beitreten:

Prüfer

Zertifizierer

Community‑Manager

Entwickler

Unterstützer

9. Kontakt / Discord‑Status
Der interne ToolAudit‑Discord ist derzeit geschlossen und nur für Team und Entwickler zugänglich.
