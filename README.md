## Hi there 👋

ToolAuditDE ist ein Community-basiertes Prüf- und Zertifizierungssystem für Third-Party-Tools wie RTSS, Magpie, 
Lossless Scaling, Upscaler, Overlays und weitere Software, die in Spielen oder produktiven Umgebungen eingesetzt wird. Ziel ist es, Transparenz, 
Sicherheit und Vertrauen zwischen Nutzern, Entwicklern und Herstellern zu schaffen.

Das offizielle ToolAudit‑Siegel kennzeichnet Tools, die durch das ToolAudit‑Team geprüft und als sicher eingestuft wurden.
Es besitzt nur dann Gültigkeit, wenn der Entwickler im internen ToolAudit‑Discord als Aktiv geführt wird.
Das Siegel zeigt an, dass das Tool die technischen und organisatorischen Anforderungen des Projekts erfüllt.

Das Community‑Siegel ist ein rein visuelles Icon, das von der Community genutzt werden kann.
Es hat keine offizielle Zertifizierungsfunktion und dient ausschließlich als Symbol für Unterstützung oder positive Resonanz.
Ein Tool gilt nicht als offiziell zertifiziert, nur weil dieses Icon verwendet wird.

Warum ToolAudit existiert
ToolAudit wurde gegründet, um Transparenz und Sicherheit im Umgang mit Third‑Party‑Tools zu schaffen. Viele Nutzer wissen nicht, ob ein Tool sicher, vertrauenswürdig oder technisch unbedenklich ist. ToolAudit bietet eine kostenlose, community‑basierte Lösung, die Vertrauen zwischen Nutzern, Entwicklern und Herstellern stärkt.

Welche Tools geprüft werden
ToolAudit konzentriert sich auf Tools wie:

RTSS

Magpie

Lossless Scaling

Upscaler

Overlays

Performance‑Tools

Grafik‑Tools

Sonstige Third‑Party‑Software, die in Spielen oder produktiven Umgebungen genutzt wird

Was ToolAudit NICHT ist
ToolAudit ist nicht:

kein Anti‑Cheat

keine Garantie für Funktionsfähigkeit

keine rechtliche Instanz

keine kommerzielle Organisation

keine Entwickler‑Bewertungsplattform

kein Ersatz für Sicherheitssoftware

ToolAudit ist ein kostenloses Community‑Projekt, das Transparenz schafft, aber keine rechtlichen oder technischen Garantien gibt.

Updates & Re‑Zertifizierung
Wenn ein Tool ein größeres Update erhält, muss die Zertifizierung erneut geprüft werden.
Nur so kann sichergestellt werden, dass das Siegel weiterhin gültig bleibt.
